Governance Model
Git-based governance
Policy Versioning
Every policy change is a commit. Every amendment is a diff. Full history, no hidden edits. 207 repos across 7 orgs on self-hosted Gitea.
Budget Transparency
Every expense tracked in budget_tracker.py. $136/mo broken down to the dollar. Variance analysis shows exactly where money goes.
Public Comments
Policy drafts have comment periods. Comments are tracked, addressed, and resolved — like pull request reviews.
FOIA by Default
Documents are public unless redacted. Redaction is tracked and auditable. All stored in the sovereign archive.
Voting Records
Every vote on every policy is recorded with timestamps and verifiable identity via decentralized ID.
Incident Accountability
Every security incident has a governance decision attached. Cron droppers, leaked PATs, miner references — all traced to root cause.
Decentralized Identity
digital_identity.py implements W3C Decentralized Identifiers (DIDs) and Verifiable Credentials. Your identity is a cryptographic key pair you control — not a username on someone else's server. Issue credentials, verify claims, resolve DIDs — all without a central authority. Backed by the same encryption that protects the fleet.

This is the foundation for everything else in Gov: budget approvals need verified identities, policy votes need authenticated participants, FOIA responses need accountable officials. Every node in the fleet has a DID. Every governance action is signed.
